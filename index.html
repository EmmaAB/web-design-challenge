This project looks at the weather conditions in over 500 different cities. Using data collected on 5 January 2017, the maximum temperature, humidity, cloudiness, and wind speed for each city were recorded along with its geographic coordinates. This allows us to look at how weather conditions vary for cities according to latitude. To view the trends side-by-side, navigate to the Comparisons page, but more indepth analysis of each condition can be found in Plots. For more information about the data represented in the Plots pages, look at the Data page.
